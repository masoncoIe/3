## 🐛 If Something Goes Wrong

### Server won't start?
```bash
# Check if port 8080 is free
lsof -i :8080

# Kill process if needed
kill -9 <PID>
```

### Site won't load?
1. Wait for "Proxy ready" status (2-3 seconds)
2. Check console (F12) for [Cole] logs
3. Try reloading the page

### Service worker error?
- Must use http://localhost:8080 (not file://)
- Can't open HTML file directly
- Must run through server

---

## 💡 Pro Tips

1. **Wait for Ready** - Don't navigate until "Proxy ready" shows
2. **First Load Slow** - Service worker needs to register (one time)
3. **Check Console** - Press F12 to see what's happening
4. **Heavy Sites** - TikTok/Instagram may take 5-10 seconds to load
5. **Reload if Stuck** - Press Ctrl+R to reload page
